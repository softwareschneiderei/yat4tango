    Class Description :
      Forwards the attributes specified by the ForwardedAttributes property. Each forwarded attribute becomes a local dynamic attribute of the AttributesForwarder device. ForwardedAttributes property is an array of strings (one entry per attribute to forward). Each entry should at least contains the fully qualified name of the attribute to be forwarded. An optional 'local attribute name' can also be specified in case you need/want to change the name under which the attribute is forwarded. A ForwardedAttributes has the following syntax: [tdb-host:tdb-port/]the/dev/name/attr-name[:local name].
